# Faith Moon Adhan Assets

Remote adhan audio assets for Faith Moon.

Use `adhan-manifest.json` as the entry point. Paths inside the manifest are relative to the repository root.

Versioned raw manifest URL:

```text
https://raw.githubusercontent.com/statweer/faith-moon-adhan-assets/v1/adhan-manifest.json
```

Notification clips are PCM CAF files intended to be downloaded into the app's local notification sound directory before scheduling. Full adhan files are AAC M4A files intended for the in-app player.
